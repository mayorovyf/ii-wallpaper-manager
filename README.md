# Менеджер обоев для ii

Это вырезка модуля обоев из моей конфигурации ii для Quickshell. Здесь только файлы, которые отвечают за выбор, предпросмотр и применение обоев.

Репозиторий не заменяет всю конфигурацию ii. Он нужен, если ты хочешь взять только менеджер обоев и подключить его в уже установленный `~/.config/quickshell/ii`.

Для работы нужны Quickshell с ii, а также `bash`, `jq`, `matugen`, `ffmpeg`, `imagemagick` и `kdialog`.

Чтобы применить файлы из этого репозитория, склонируй его и выполни синхронизацию:

```bash
rsync -a .config/quickshell/ii/ ~/.config/quickshell/ii/
```

После этого перезапусти Quickshell.

По умолчанию модуль смотрит обои в `~/Wallpapers`. Если у тебя другой путь, поменяй `defaultFolder` в `services/Wallpapers.qml`.
